# C50-Project-Python
Projects completed as part of Harvard CS50 course, focus on problem-solving, algorithms, and programming in Python

1. Cash (cash.py)
Overview

Calculates the minimum number of coins needed to make change for a given amount in dollars.

Features
Prompts user for a monetary amount.
Calculates coins using quarters, dimes, nickels, and pennies.
Ensures minimum number of coins.

2. Credit (credit.py)
Overview

Validates credit card numbers and identifies card type (VISA, AMEX, MasterCard).

Features
Uses Luhn’s Algorithm for validation.
Accepts numeric input from the user.
Prints card type or “INVALID” if invalid.

3. DNA (dna.py)
Overview

Detects a person based on a DNA sequence and a database of known STR (Short Tandem Repeat) counts.

Features
Reads DNA sequence from a text file.
Compares STR counts against a CSV database.
Prints matching individual’s name or “No match”.

4. Mario (More & Less) (mario-more.py, mario-less.py)
Overview

Python version of the CS50 Mario pyramid programs. Prints half or full pyramids of hashes.

Features
Prompts user for pyramid height (1–8).
mario-less.py prints a left-aligned half-pyramid.
mario-more.py prints a full pyramid with spaces and hashes.
5. Phonebook CSV (phonebookcsv.py)
Overview

Manages a phonebook using CSV files. Allows adding, searching, and listing contacts.

Features
Stores contacts in contacts.csv.
Supports interactive user input.
Reads and writes data using Python’s csv module.

Overview
6. Readability (readability.py)
Overview

Analyzes text for reading grade level using the Coleman-Liau index.

Features
Counts letters, words, and sentences.
Computes and prints grade level.
Handles multiple sentences and punctuation.

Notes for All Python Projects
Written in Python 3.
Follows CS50 2026 problem set requirements.
Uses standard libraries (csv, sys) where necessary.
Includes input validation and robust error handling.
